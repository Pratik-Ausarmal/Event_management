# Event Management Website

A complete event management system built with Node.js, Express, MySQL, and Bootstrap.

## Features

- User Authentication (Registration, Login, Role-based access)
- Event Management (Create, View, Book Events)
- Multi-step Booking System with Service Selection
- User Dashboard for managing bookings
- Admin Panel for system management
- Responsive Bootstrap Design
- Image Gallery
- Contact Form
- Testimonials Section
